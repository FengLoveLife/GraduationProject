## 关键决策

### 架构设计

```
前端(Vue) → Java后端(Spring Boot) → Python服务(FastAPI) → LightGBM模型
```

### 技术选型

- **模型**：LightGBM（轻量、快速、适合表格数据）
- **Python框架**：FastAPI（类似Spring MVC，易上手）
- **特征工程**：简化版（lag_1, lag_7, rolling_7_mean等）

### 简化策略

为了在20天内完成，采用以下简化：
1. 特征工程只使用基础特征，不做复杂工程
2. 模型使用默认参数，不做深度调优
3. 先跑通再优化，不追求工业级完美
