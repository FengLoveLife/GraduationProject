### 4. 品类销售占比饼图修复

**问题**：饼图显示"未知占比"，无法显示品类名称

**原因分析**：
- `sales_order_item.category_name` 字段为 NULL
- 数据导入时未填充该字段
- 但 `category_id` 有值

**解决方案**：
修改 `SalesOrderItemMapper.getCategoryPie()` SQL：
```sql
-- 通过二级分类关联一级分类
SELECT COALESCE(pc1.name, '未分类') as categoryName, SUM(soi.subtotal_amount) as amount
FROM sales_order_item soi
LEFT JOIN product_category pc2 ON soi.category_id = pc2.id
LEFT JOIN product_category pc1 ON pc2.parent_id = pc1.id
WHERE soi.sale_date BETWEEN #{startDate} AND #{endDate}
GROUP BY categoryName ORDER BY amount DESC
```

**效果**：按5个一级分类显示（粮油调味、日用百货、生鲜果蔬、酒水饮料、休闲零食）

## 技术要点记录

### 一级分类关联查询

商品分类采用两级结构：
- 一级分类：parent_id = 0, level = 1（如：休闲零食、酒水饮料）
- 二级分类：parent_id = 一级分类ID, level = 2（如：碳酸饮料、坚果炒货）

销售明细存储的是二级分类ID，查询一级分类需要两次JOIN：
```sql
sales_order_item.category_id -> product_category(pc2) -> product_category(pc1.parent_id)
```

### 日历组件设计

使用 CSS Grid 手动绘制42格日历（6行×7列），而非 Element Plus 的 el-calendar 组件：
- 更灵活的样式控制
- 支持自定义内容（天气图标、节假日标签）
- 更好的交互体验
