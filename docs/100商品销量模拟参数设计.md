# 100商品销量模拟参数设计文档

> 本文档详细说明中小型超市100个商品的销售特性及影响因子参数设计
> 所有参数基于真实超市运营数据规律设计，确保模拟数据具有现实意义

---

## 一、影响因子说明

### 1.1 星期因子（weekday_factor）

中小型超市的客流规律：
- **周一至周四**：工作日，客流较少，销量平稳
- **周五**：工作日尾声，客流开始回升
- **周六、周日**：周末，客流高峰，销量明显上升

**典型星期因子设计原则：**
```
周一(1): 0.85-1.0    (工作日低谷)
周二(2): 0.85-1.0
周三(3): 0.85-1.0
周四(4): 0.90-1.0
周五(5): 1.0-1.15    (回升)
周六(6): 1.15-1.50   (周末高峰)
周日(7): 1.15-1.50
```

### 1.2 天气因子（weather_factor）

天气编码：0=晴，1=多云，2=雨，3=炎热

**天气对销量的影响规律：**

| 天气 | 影响商品 | 影响方向 | 因子范围 |
|------|---------|---------|---------|
| 晴(0) | 正常 | 基准 | 1.0 |
| 多云(1) | 正常 | 略降 | 0.95-1.0 |
| 雨(2) | 多数商品 | 客流减少 | 0.7-0.9 |
| 炎热(3) | 冷饮、啤酒、冰淇淋 | 大幅上升 | 1.5-2.5 |
| 炎热(3) | 生鲜、蔬菜 | 易损耗降 | 0.8-0.9 |

### 1.3 节假日因子（holiday_factor）

节假日编码：0=普通日，1=节假日

**节假日对销量的影响规律：**

| 商品类型 | 节假日影响 | 因子范围 |
|---------|-----------|---------|
| 休闲零食 | 走亲访友、聚会增加 | 1.3-1.8 |
| 酒水饮料 | 节庆饮酒增加 | 1.2-1.5 |
| 生鲜果蔬 | 节前囤货，节中下降 | 1.0-1.5 |
| 粮油调味 | 节前囤货明显 | 1.2-2.0 |
| 日用百货 | 影响较小 | 1.0-1.1 |

### 1.4 基础销量（base_sales）

基于商品定价、库存周转率、超市规模（中小型）综合估算：

| 商品类型 | 日均销量范围 | 依据 |
|---------|-------------|------|
| 高频日用品（水、可乐） | 30-80件/天 | 消费高频、单价低 |
| 常用食品（牛奶、零食） | 15-40件/天 | 日常消费 |
| 中频商品（粮油、洗护） | 5-15件/天 | 周期性购买 |
| 低频高价商品（进口食品） | 2-8件/天 | 单价高、需求少 |
| 生鲜类（蔬菜、水果） | 10-30kg/天 | 家庭日常需求 |

## 四、数据模拟建议

### 4.1 随机波动设置

建议设置 **±15%** 的随机噪声：
```python
noise = np.random.uniform(-0.15, 0.15)
sales = base_sales * weekday_factor * weather_factor * holiday_factor * (1 + noise)
```

### 4.2 天气生成规则

- 晴天(0): 40%概率
- 多云(1): 25%概率
- 雨(2): 20%概率
- 炎热(3): 15%概率
- 夏季(6-8月): 炎热概率提升至30-40%

### 4.3 节假日配置

建议覆盖以下节假日：
- 清明节(4月初)
- 劳动节(5月1-3日)
- 端午节(农历五月初五)
- 建党节(7月1日)
- 中秋节(农历八月十五)
- 国庆节(10月1-7日)
