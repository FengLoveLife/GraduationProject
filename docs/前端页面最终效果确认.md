## 七、接口需求汇总

### 7.1 销量预测接口

```
POST /api/forecast/predict/batch
请求参数：
{
  "date": "2026-03-28",
  "weather": "晴",      // 可选
  "isHoliday": false    // 可选
}

响应：
{
  "code": 200,
  "data": [
    {
      "productId": 1,
      "productCode": "SP001",
      "productName": "可口可乐500ml",
      "categoryId": 21,
      "categoryName": "饮料",
      "predictedQuantity": 45,
      "currentStock": 30,
      "safetyStock": 20
    },
    ...
  ]
}
```

### 7.2 进货建议接口

```
GET /api/purchase/suggestion/list
响应：
{
  "code": 200,
  "data": [
    {
      "id": 1,
      "productId": 1,
      "productCode": "SP001",
      "productName": "可口可乐500ml",
      "categoryName": "饮料",
      "purchasePrice": 2.50,
      "predictedQuantity": 45,
      "currentStock": 30,
      "safetyStock": 20,
      "targetStock": 85,
      "dailySales": 5,
      "lightStatus": 1,        // 1=红灯 2=黄灯
      "suggestedQuantity": 35,
      "finalQuantity": 35
    },
    ...
  ]
}
```

### 7.3 生成进货单接口

```
POST /api/purchase/order/create
请求参数：
{
  "suggestionIds": [1, 2, 3],

  "remark": "备注信息"
}

响应：
{
  "code": 200,
  "data": {
    "orderId": 1,
    "orderNo": "PO-20260328-001"
  }
}
```
