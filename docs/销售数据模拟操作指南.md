## 二、数据模拟规则详解

### 2.1 核心公式

```
最终销量 = 基础销量 × 时间因子 × 天气因子 × 节假日因子 × 促销因子 × 随机扰动
```

### 2.2 各因子详细规则

#### A. 时间因子（星期规律）

| 星期 | 因子值 | 说明 |
|------|--------|------|
| 周一 | 0.85 | 一周开始，客流较少 |
| 周二 | 0.90 | 工作日 |
| 周三 | 0.90 | 工作日 |
| 周四 | 0.95 | 接近周末 |
| 周五 | 1.10 | 周末前夕 |
| 周六 | 1.40 | 周末高峰 |
| 周日 | 1.50 | 周末最高峰 |

#### B. 天气因子

| 天气 | 整体因子 | 特殊品类调整 |
|------|----------|--------------|
| 晴天 | 1.0 | 饮料类 ×1.3，水果类 ×1.2 |
| 多云 | 1.0 | 无调整 |
| 小雨 | 0.85 | 雨具类 ×1.5 |
| 大雨 | 0.60 | 饮料类 ×0.7，雨具类 ×2.0 |
| 炎热(>30°C) | 1.1 | 饮料类 ×2.0，水果类 ×1.5 |
| 寒冷(<5°C) | 0.9 | 热饮类 ×1.5 |

#### C. 节假日因子

| 日期类型 | 因子值 | 说明 |
|----------|--------|------|
| 普通日 | 1.0 | 正常销售 |
| 节假日前夕 | 1.3 | 采购高峰 |
| 节假日当天 | 1.5 | 客流高峰 |
| 春节等大节 | 2.0 | 超级高峰 |

#### D. 促销因子

| 是否促销 | 因子值 | 说明 |
|----------|--------|------|
| 正常销售 | 1.0 | 无促销 |
| 促销商品 | 3.0~5.0 | 随机促销力度 |

#### E. 随机扰动

- 范围：±10%~20%
- 使用正态分布生成，模拟真实的不确定性
- 这样每天的销量会有波动，但整体趋势不变

---

### 2.3 品类差异

不同品类的基础销量不同：

| 品类 | 日均基础销量 | 特殊规律 |
|------|--------------|----------|
| 饮料/饮品 | 15-30 | 天气敏感 |
| 休闲零食 | 10-20 | 周末效应明显 |
| 粮油调味 | 5-10 | 稳定，节假日略高 |
| 日用百货 | 8-15 | 稳定 |
| 新鲜水果 | 12-25 | 天气敏感，易损耗 |
| 乳制品 | 10-18 | 稳定 |

## 七、常见问题

### Q1: pip 安装报错怎么办？
尝试升级 pip：
```bash
python -m pip install --upgrade pip
```

### Q2: 商品数据怎么导出？
使用 Navicat/DataGrip 等工具，或告诉我，我帮你写一个 SQL 脚本直接导出 CSV。

### Q3: 生成的数据能修改吗？
可以！脚本中有参数可以调整，比如：
- `TOTAL_DAYS`：模拟天数
- `BASE_SALES_RANGE`：基础销量范围
- `RANDOM_NOISE`：随机扰动幅度
